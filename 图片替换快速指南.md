## 🚀 快速开始（10分钟）

### 第1步：获取3张插画（5分钟）

#### 1.1 打开网站
访问：https://undraw.co

#### 1.2 下载作业场景插画
1. 搜索 "homework" 或 "study"
2. 选择一个合适的插画
3. 点击右侧 "Change Color"
4. 输入颜色代码：`#B5A8A8`
5. 点击 "Download PNG"
6. 保存命名为：`homework.png`

#### 1.3 下载磨蹭场景插画
1. 搜索 "clock" 或 "morning"
2. 选择一个合适的插画
3. 改色为：`#A8B5B5`
4. 下载命名为：`procrastination.png`

#### 1.4 下载顶嘴场景插画
1. 搜索 "discussion" 或 "argument"
2. 选择一个合适的插画
3. 改色为：`#B5C1AA`
4. 下载命名为：`talkback.png`

### 第2步：获取小程序码（3分钟）

#### 2.1 登录微信公众平台
访问：https://mp.weixin.qq.com/

#### 2.2 生成小程序码
1. 点击左侧"开发"
2. 点击"开发设置"
3. 找到"小程序码"区域
4. 点击"获取"
5. 路径填写：`pages/index/index`
6. 点击"生成"
7. 下载命名为：`qrcode.png`

### 第3步：保存到项目（1分钟）

将4张图片保存到：
```
/Users/guojiaqi/Documents/半糖的claude code/正念育儿小程序/images/

share-cards/
  ├── homework.png
  ├── procrastination.png
  └── talkback.png

qrcode.png
```

### 第4步：测试（1分钟）

1. 在微信开发者工具点击"编译"
2. 进入任意场景完成呼吸练习
3. 点击"分享给朋友"
4. 查看插画是否显示

## 🆘 常见问题

### Q1: unDraw.co 打不开？
**A**: 尝试以下网站：
- Storyset.com
- Icons8.com/illustrations
- Illlustrations.co

### Q2: 下载的图片不清晰？
**A**:
- 确保选择 PNG 格式
- 选择 800px 以上尺寸
- unDraw 下载时选择 "High Quality"

### Q3: 颜色怎么改？
**A**:
- unDraw: 右侧 "Change Color" 按钮
- 输入对应的莫兰迪色值即可

### Q4: 小程序码在哪里？
**A**:
- 必须在微信公众平台生成
- 需要管理员权限
- 如果没有权限，暂时用占位图也可以
