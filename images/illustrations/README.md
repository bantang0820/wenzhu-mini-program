# 正念日记插图资源说明

## 📌 插图需求

正念日记卡片功能需要以下插图资源，用于美化卡片视觉效果。

## 🎨 需要的插图文件

### 1. calm.png - 平静插图
- **用途**：孩子发脾气场景
- **风格**：温暖、治愈
- **尺寸**：建议 400x400px
- **建议内容**：
  - 父母温柔拥抱孩子的画面
  - 或平静呼吸的插画
  - 色调温暖（暖黄、米色）

### 2. love.png - 爱的插图
- **用途**：冷战场景
- **风格**：温馨、连接
- **尺寸**：建议 400x400px
- **建议内容**：
  - 心形元素
  - 或手牵手画面
  - 色调柔和（粉色、桃色）

### 3. peaceful.png - 宁静插图
- **用途**：早上匆忙场景
- **风格**：宁静、有序
- **尺寸**：建议 400x400px
- **建议内容**：
  - 晨光、阳光元素
  - 或安静喝咖啡的画面
  - 色调清新（浅蓝、米白）

### 4. sleep.png - 睡眠插图
- **用途**：睡前大战场景
- **风格**：安静、安稳
- **尺寸**：建议 400x400px
- **建议内容**：
  - 月亮、星星元素
  - 或温馨的睡前画面
  - 色调柔和（深蓝、紫色）

### 5. mindful.png - 正念通用插图
- **用途**：默认场景
- **风格**：正念、冥想
- **尺寸**：建议 400x400px
- **建议内容**：
  - 莲花、禅意元素
  - 或冥想的剪影
  - 色调自然（绿色、米色）

## 📁 文件位置

将插图文件放置在：
```
/images/illustrations/
├── calm.png
├── love.png
├── peaceful.png
├── sleep.png
└── mindful.png
```

## 🎯 设计建议

1. **统一风格**：所有插图保持相同的绘画风格
2. **配色方案**：使用小程序主色调（金色 #D4AF37、米色 #F5F3EF）
3. **简洁大方**：不要太复杂，确保在小卡片上清晰可见
4. **情感表达**：传递平静、温暖、治愈的感觉

## 📐 技术要求

- **格式**：PNG（支持透明背景）
- **尺寸**：400x400px 或更大
- **文件大小**：每个图片建议 < 200KB
- **透明背景**：建议使用透明背景，便于融合到卡片中

## 🎨 获取方式

### 方式一：自己设计
使用设计工具（Canva、Figma、Photoshop）设计

### 方式二：购买素材
推荐素材网站：
- 花瓣网
- 站酷海洛
- Undraw（免费插画）
- Storyset（免费动画风插画）

### 方式三：使用占位图
开发期间可以先用纯色块占位，后续替换为真实插图

## ⚠️ 注意事项

1. **版权问题**：确保使用的插图有合法使用权
2. **文件命名**：严格按照上述命名，不要修改
3. **图片质量**：确保图片清晰，不模糊
4. **测试效果**：在不同设备上测试显示效果

## 🔄 更新说明

创建好插图后，直接放入对应目录即可，无需修改代码。

代码会自动根据场景ID加载对应的插图。
