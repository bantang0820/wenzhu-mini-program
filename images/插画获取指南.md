# 🎨 免费插画获取指南

## 快速获取方案（推荐）⭐

### 方案A：unDraw（最快，5分钟）

#### 步骤：
1. 访问 https://undraw.co
2. 点击搜索图标，搜索关键词：
   - "homework" → 作业场景
   - "clock" / "time" → 磨蹭场景
   - "conversation" / "talking" → 顶嘴场景
3. 点击选中的插画
4. 点击右侧 "Change Color" 按钮
5. 输入莫兰迪色值：
   - #B5A8A8（红灰）
   - #A8B5B5（蓝灰）
   - #B5C1AA（绿灰）
6. 点击下载 PNG 格式
7. 保存到 `images/share-cards/` 目录：
   - 作业：`homework.png`
   - 磨蹭：`procrastination.png`
   - 顶嘴：`talkback.png`

#### 优点：
✅ 完全免费，可商用
✅ 可自定义配色
✅ 高质量PNG
✅ 5分钟搞定

### 方案B：Storyset（动画风格，10分钟）

#### 步骤：
1. 访问 https://storyset.com
2. 搜索关键词：
   - "Woman studying"（学习场景）
   - "Clock alarm"（时间场景）
   - "Discussion"（对话场景）
3. 点击插画进入编辑器
4. 调整配色为莫兰迪色系
5. 点击右上角 "Download" → "PNG"
6. 保存到对应目录

#### 优点：
✅ 插画更生动
✅ 支持动画
✅ 现代风格

---

### 方案C：Icons8（扁平风格，5分钟）

#### 步骤：
1. 访问 https://icons8.com/illustrations
2. 搜索关键词：
   - "education"（教育）
   - "late"（迟到）
   - "argument"（争吵）
3. 选择 "Flat" 风格
4. 点击 "Download" → 250px
5. 免费注册后下载

#### 优点：
✅ 风格统一
✅ 高质量
✅ 可商用

## 🎯 具体操作步骤（最简单）

### 我推荐用 unDraw，步骤如下：

#### 1. 作业场景插画
```
1. 打开 https://undraw.co
2. 搜索 "homework" 或 "study"
3. 选择一个孩子在书桌前的插画
4. 点击插画，右侧面板
5. "Change Color" → 输入 #B5A8A8
6. 点击 "Download PNG"
7. 重命名为 "homework.png"
8. 保存到项目的 images/share-cards/ 目录
```

#### 2. 磨蹭场景插画
```
1. 在 undraw.co 搜索 "morning" 或 "clock"
2. 选择一个有时钟/早晨元素的插画
3. 改色为 #A8B5B5
4. 下载命名为 "procrastination.png"
5. 保存到 images/share-cards/
```

#### 3. 顶嘴场景插画
```
1. 在 undraw.co 搜索 "discussion" 或 "argument"
2. 选择一个对话/争论场景的插画
3. 改色为 #B5C1AA
4. 下载命名为 "talkback.png"
5. 保存到 images/share-cards/
```

## 🖼️ 图片规格要求

### 文件信息
- **格式**：PNG（透明背景）
- **尺寸**：800 x 600 px 或以上
- **文件大小**：建议 < 500KB
- **背景**：透明最好

### 命名规范
```
images/share-cards/
├── homework.png          # 作业场景
├── procrastination.png   # 磨蹭场景
└── talkback.png          # 顶嘴场景
```
