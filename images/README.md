## 🖼️ 需要添加的图片

### 1. 分享卡片插画（3张）

请参考《插画获取指南.md》获取免费可商用的插画：

#### homework.png（作业场景）
- **尺寸**：800 x 600 px
- **格式**：PNG（透明背景）
- **风格**：扁平莫兰迪风
- **配色**：#B5A8A8（红灰）
- **场景**：孩子写作业、书桌、学习相关

#### procrastination.png（磨蹭场景）
- **尺寸**：800 x 600 px
- **格式**：PNG（透明背景）
- **风格**：扁平莫兰迪风
- **配色**：#A8B5B5（蓝灰）
- **场景**：时钟、早晨、匆忙相关

#### talkback.png（顶嘴场景）
- **尺寸**：800 x 600 px
- **格式**：PNG（透明背景）
- **风格**：扁平莫兰迪风
- **配色**：#B5C1AA（绿灰）
- **场景**：对话、争论、沟通相关

### 2. 小程序码（1张）

#### qrcode.png
- **尺寸**：500 x 500 px
- **格式**：PNG
- **获取方式**：
  1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
  2. 进入"开发" → "开发设置"
  3. 点击"生成小程序码"
  4. 路径填写：`pages/index/index`
  5. 下载并保存为 `qrcode.png`
